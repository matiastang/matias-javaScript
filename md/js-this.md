# this

JavaScript 的一大特点是，函数存在「定义时上下文」和「运行时上下文」以及「上下文是可以改变的」这样的概念。
[MDN this](https://developer.mozilla.org/zh-cn/docs/web/javascript/reference/operators/this)

在箭头函数中，this与封闭词法环境的this保持一致。
